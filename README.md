## Project Title: Page Visits Funnel Project

The purpose of this project was to analyze data on visits to a retail website. The goal was to build a funnel––a description of the number of visitors that continue to the next step of the multi-step process: site visit, placement of item in cart, checkout, and purchase. The funnel was then used to determine the weakest step in the multi-step process.

### The project includes four datasets

- visits.csv: id and timestamp of all users who have visited the website
- cart.csv: id and timestamp of all users who added an item(s) to their cart
- checkout.csv: id and timestamp of all users who proceeded the checkout
- purchase.csv: id and timestam of all users who made a purchase
